# Quick Start Guide - Register Under Udyam in 5 Minutes

**Fastest way to get your Udyam Registration**

---

## ⏱️ 5-Minute Process

### Minute 1: Prepare Documents
```
✅ Aadhaar Card (mandatory)
✅ Mobile Number (for OTP)
✅ Bank Account Details
✅ Business Details (name, type, location)
```

### Minute 2: Visit Official Portal
```
🌐 Go to: https://udyamregistration.gov.in/
🖱️ Click: "Register as Udyam"
```

### Minute 3: Fill Basic Information
```
📝 Organization Type: (Sole Proprietor/Partnership/Company)
📝 Business Name: [Your Business]
📝 State & District: [Your Location]
📝 NIC Code: [Business Type]
```

### Minute 4: Add Investment & Bank Details
```
💰 Investment in Plant & Machinery: [Amount]
🏦 Bank Name: [Your Bank]
🏦 Account Number: [Your Account]
🏦 IFSC Code: [Bank IFSC]
```

### Minute 5: Verify & Submit
```
📱 Receive OTP on mobile
✓ Enter OTP
✓ Submit Application
🎉 Instant Certificate Generated!
```

---

## 🎯 What You Get Instantly

✅ **Udyam Aadhaar Number** - Your unique ID  
✅ **E-Certificate** - Digital certificate  
✅ **Lifetime Validity** - No renewal needed  
✅ **Government Recognition** - Official status  
✅ **Loan Eligibility** - Up to ₹1 crore collateral-free

## 💡 Quick Tips

### ✅ DO
- Use correct Aadhaar linked mobile
- Have exact bank details ready
- Choose correct NIC code
- Double-check investment amount

### ❌ DON'T
- Close browser before submission
- Use old mobile number
- Guess investment amount
- Make typos in business name

---

## 🆘 If Something Goes Wrong

| Problem | Solution |
|---------|----------|
| OTP not received | Check mobile linked to Aadhaar |
| Bank details error | Verify IFSC code and account number |
| NIC code error | Use NIC finder on portal |
| Investment rejected | Ensure it includes only eligible items |
