# Udyam Registration Documents Checklist

**Complete list of documents needed for registration**

---

## ✅ Essential Documents (MANDATORY)

### Personal Identification
- [ ] **Aadhaar Card Number** (must be linked to current mobile)
- [ ] **Valid Mobile Number** (for OTP verification)
- [ ] **Email Address** (for certificate delivery)

### Financial Information
- [ ] **Bank Account Number**
- [ ] **IFSC Code** (11-digit bank code)
- [ ] **Account Holder Name** (must match Aadhaar)
- [ ] **Bank Name & Branch**

### Business Information
- [ ] **Business Name** (as you want to register)
- [ ] **Business Type** (Sole Proprietor/Partnership/Company)
- [ ] **NIC Code** (4-digit industry code)
- [ ] **Business Address** (physical or registered office)
- [ ] **Investment Amount** (in plant & machinery)

---

## 📋 Recommended Documents (NON-MANDATORY BUT HELPFUL)

### Tax/Government Documents
- [ ] **PAN Card** (for income tax purposes)
- [ ] **Income Tax Returns** (last 2 years, if applicable)
- [ ] **GST Certificate** (if already registered)

### Business Registration Documents
- [ ] **Shop Establishment Certificate** (if applicable)
- [ ] **Trade License** (if applicable)
- [ ] **Business License**
- [ ] **Partnership Deed** (if partnership)
- [ ] **Company Registration Certificate** (if company)

### Property Documents
- [ ] **Property Ownership Proof** (if owned)
- [ ] **Lease Deed** (if rented)
- [ ] **Tenancy Agreement**

### Financial Documents
- [ ] **Bank Statements** (last 6-12 months)
- [ ] **Profit & Loss Statement** (if existing business)
- [ ] **Balance Sheet** (if applicable)

## 🔑 How to Get Required Documents

### Aadhaar Card
- Already have? ✅ Just note the number
- Don't have? Visit nearest Aadhaar enrollment center
- **Time**: 1 day (instant in some centers)

### Bank Details
- Check your:
  - Checkbook (has account number)
  - Bank statement
  - Bank passbook
  - Bank app
- **What you need**:
  - Account number (10-18 digits)
  - IFSC code (11 characters, starts with bank code)

### PAN Card
- Already have? ✅ Just note the number
- Don't have? Apply online at [incometaxindiaefiling.gov.in](https://incometaxindiaefiling.gov.in/)
- **Time**: 5-10 days

### Business Registration Certificates
- Contact your municipal corporation
- Or local government office
- **Cost**: ₹500-₹2000

## ⚠️ Important Verification Rules

### Aadhaar Requirements
- ✅ Must be valid and active
- ✅ Must be linked to current mobile number
- ⚠️ If not linked:
  1. Visit Aadhaar enrollment center
  2. Update mobile number
  3. Wait for confirmation
  4. Then proceed with registration

### Bank Account Requirements
- ✅ Account must be in your name
- ✅ Account must be active
- ✅ Account holder name must match Aadhaar name
- ⚠️ If name doesn't match:
  1. Update bank account holder name
  2. Or update Aadhaar name
  3. Wait for confirmation
  4. Then proceed

### Name Consistency
- ✅ Aadhaar name
- ✅ PAN name (if applicable)
- ✅ Bank account name
- All should MATCH
- If not, get them corrected first

## 📝 Document Preparation Checklist

### Before Registration
- [ ] Aadhaar card number noted
- [ ] Mobile number linked to Aadhaar verified
- [ ] Bank account details confirmed
  - [ ] Account number
  - [ ] IFSC code
  - [ ] Account holder name
  - [ ] Bank name
- [ ] Business name finalized
- [ ] NIC code identified
- [ ] Investment amount calculated
- [ ] Aadhaar-PAN-Bank name consistency checked

### During Registration
- [ ] All details entered correctly
- [ ] No typos in names or numbers
- [ ] IFSC code correct (check format: 4 letters + 0 + 6 digits)
- [ ] Investment amount verified

### After Registration
- [ ] OTP received and verified
- [ ] Certificate downloaded
- [ ] Screenshot taken for backup
- [ ] Udyam Aadhaar number noted

## 🆘 If Documents Are Missing or Wrong

### Missing Aadhaar Link
✅ **Solution**: Visit Aadhaar enrollment center immediately  
⏱️ **Time**: Same day

### Wrong Bank Details
✅ **Solution**: Can update within 30 days of registration  
⏱️ **Time**: Log in to portal and update

### Name Mismatch
✅ **Solution**: Correct source document before registration  
⏱️ **Time**: 3-5 days

### Investment Amount Query
✅ **Solution**: Include all eligible items (machinery, equipment, tools)  
📋 **What to do**: Recalculate and update if within 30 days
