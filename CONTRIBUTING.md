# Contributing to Instant Udyam

**Help us build better resources for Indian MSMEs**

---

## 📚 How You Can Contribute

### 1. Documentation Improvements
- Suggest corrections or clarifications
- Add new guides or tutorials
- Improve examples and explanations

### 2. Content Creation
- Write case studies of successful Udyam businesses
- Share your Udyam registration experience
- Create templates and checklists

### 3. Language Support
- Help translate guides to Indian languages
- Adapt content for regional needs
- Localize schemes information

### 4. Code & Tools
- Build NIC code finder tools
- Create calculators (investment, loan eligibility)
- Develop tracking utilities

### 5. Feedback & Issues
- Report inaccuracies
- Suggest new topics
- Share improvement ideas

## 🚀 Getting Started

### Fork the Repository
```bash
git clone https://github.com/your-username/Instant-Udyam.git
cd Instant-Udyam
```

### Create a Feature Branch
```bash
git checkout -b feature/your-contribution
```

### Make Your Changes
- Edit or create documentation
- Add code improvements
- Update resources

### Commit Your Work
```bash
git add .
git commit -m "Add: Description of your contribution"
```

### Push and Create Pull Request
```bash
git push origin feature/your-contribution
```
Then create a pull request on GitHub

## 📋 Contribution Guidelines

### Documentation Standards
- Use clear, simple language
- Include examples and scenarios
- Add visual separators (emojis, tables)
- Link to official resources
- Verify government information

### Content Quality
- ✅ Accurate and up-to-date
- ✅ Well-structured with headers
- ✅ Complete with examples
- ✅ Easy to understand
- ✅ Helpful for MSMEs

### Code Standards
- Clear variable names
- Comments for complex logic
- Follow Python/JavaScript conventions
- Test before submitting

## 📝 Pull Request Process

1. Fork the repository
2. Create feature branch
3. Make changes
4. Ensure quality and accuracy
5. Submit PR with description
6. Address review feedback
7. Merge after approval
